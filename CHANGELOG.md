# Changelog

All notable changes to this project should be documented here.

This project follows a simple human-readable changelog format.

## v1.0.0 - 2026-06-27

- Convert the Java contacts application into a Maven monorepo with `core`, `cli`, and `api` modules.
- Add a Next.js web dashboard for managing contacts through the Java API.
- Add Docker images for the API and web dashboard.
- Add AWS Terraform infrastructure for ECR, ECS Fargate, ALB, EFS, CloudWatch, and networking.
- Add GitHub Actions CI, CodeQL, Terraform validation, deployment, and release workflows.
- Add repository-wide formatting, Git hooks, documentation, and contribution policy.
